#무엇을만드는가
자동으로 강아지의 변을 치워주고 변의 상태를 진단해주는 기기를 만듭니다.

#왜 만들어야 했을까요?
변을 치우는게 번거롭기 때문이죠. 하지만 견주로서 변의 상태를 확인할 필요는 있기때문에 그것을 기계가 대신해주도록 하고자 합니다.
